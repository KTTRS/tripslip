# TripSlip Shared Packages

This directory contains shared packages used across all TripSlip applications:

- **ui** - Shared Radix UI components and design system
- **database** - Supabase client, TypeScript types, and database utilities
- **auth** - Authentication utilities and session management
- **i18n** - Internationalization configuration and translations (EN/ES/AR)
- **utils** - Shared utility functions (validation, formatting, error handling)

Each package is independently versioned and can be imported by any application in the monorepo.
