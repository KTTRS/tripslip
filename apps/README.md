# TripSlip Applications

This directory contains the five separate web applications that make up the TripSlip platform:

- **landing** - Public marketing website (tripslip.com)
- **venue** - Venue management application (venue.tripslip.com)
- **school** - School/district admin application (school.tripslip.com)
- **teacher** - Teacher trip management application (teacher.tripslip.com)
- **parent** - Parent permission slip application (parent.tripslip.com)

Each application is a separate React SPA built with Vite and TypeScript, sharing common packages from the `packages/` directory.
