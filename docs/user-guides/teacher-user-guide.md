# Teacher User Guide

**TripSlip Teacher App** - Complete guide for teachers planning field trips

## Getting Started

### Quick Start (No Account Required)

1. Visit `teacher.tripslip.com`
2. Click "Plan a Trip" to start immediately
3. No login required for basic trip planning

### Creating an Account (Optional)

Benefits of creating an account:
- Save trips and access them later
- Track permission slips and payments
- Manage multiple trips
- Access trip history

## Planning a Trip

### Step 1: Browse Experiences

1. Search by location, date, or grade level
2. Filter by subject area or educational standards
3. View experience details and pricing
4. Read reviews from other teachers

### Step 2: Create Trip

1. Select an experience
2. Choose trip date and time
3. Enter student count
4. Add optional add-ons
5. Review total cost

### Step 3: Add Student Roster

**Option 1: Manual Entry**
- Enter student names one by one
- Add parent email addresses
- Include emergency contacts

**Option 2: CSV Import**
- Download CSV template
- Fill in student information
- Upload completed file
- Review and confirm data

### Step 4: Generate Permission Slips

1. Review trip details
2. Customize permission slip message
3. Click "Generate Permission Slips"
4. System sends magic links to parents

## Managing Permission Slips

### Tracking Status

Dashboard shows real-time status:
- **Pending**: Not yet signed
- **Signed**: Permission granted
- **Paid**: Payment completed (if required)
- **Declined**: Parent declined permission

### Sending Reminders

2. Click "Send Reminder"
3. Parents receive reminder email
4. Track reminder history

## Troubleshooting

### Common Issues

**Parents not receiving permission slips**
- Verify email addresses are correct
- Check spam/junk folders
- Resend permission slip
- Provide direct link

**CSV import errors**
- Use provided template
- Check for special characters
- Verify email format
- Remove empty rows

**Payment not showing as received**
- Allow 24 hours for processing
- Check with parent for confirmation
- Contact support if issue persists

## FAQ

**Do I need an account to use TripSlip?**
No, you can plan trips without an account. However, creating an account lets you save and manage multiple trips.

**How do parents access permission slips?**
Parents receive a secure magic link via email. No account or password required.

**What if a parent doesn't have email?**
You can print a paper permission slip or provide the magic link via text message.

**Can I edit trip details after sending permission slips?**
Yes, but parents will be notified of changes. Major changes may require new permission slips.

**How do I handle students without permission?**
Students without signed permission slips cannot attend. Work with school administration for alternatives.

**Is student data secure?**
Yes, TripSlip is FERPA compliant and uses industry-standard security measures to protect student information.
