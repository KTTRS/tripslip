# School Administrator User Guide

**TripSlip School App** - Complete guide for school and district administrators

## Getting Started

### Account Setup

1. Visit `school.tripslip.com`
2. Create administrator account
3. Verify email address
4. Complete school profile

### School Profile

Configure:
- School/district name
- Address and contact information
- School type and grade levels
- Number of students and teachers
- District policies

## Managing Teachers

### Inviting Teachers

1. Navigate to "Teachers" section
2. Click "Invite Teacher"
3. Enter teacher email and name
4. Select permissions level
5. Send invitation

### Teacher Permissions

- **Standard**: Create trips, manage own trips
- **Lead**: Approve trips, view all school trips
- **Admin**: Full access, manage teachers

### Teacher Management

- View all teachers
- Edit teacher information
- Deactivate accounts
- Reset passwords
- View teacher activity

## Troubleshooting

### Common Issues

**Teacher can't access system**
- Verify invitation sent
- Check email address
- Resend invitation
- Reset password

**Trip approval not working**
- Check approval workflow settings
- Verify permissions
- Review trip details
- Contact support

**Budget tracking incorrect**
- Verify trip costs entered correctly
- Check budget allocation
- Reconcile with financial system
- Update budget as needed

## FAQ

**How many teachers can we add?**
Unlimited teachers can be added to your school account.

**Can we customize permission slip templates?**
Yes, you can customize templates to match your school's requirements and branding.

**How is student data protected?**
TripSlip is FERPA compliant with industry-standard security measures and regular audits.

**Can we integrate with our SIS?**
Yes, TripSlip offers integration options with major student information systems.

**What reports are available?**
Comprehensive reports including trip summaries, budget tracking, participation rates, and custom reports.

**How do we handle financial assistance?**
Configure payment policies to accommodate financial assistance programs and work with teachers on individual cases.

**Can we set different policies for different grade levels?**
Yes, policies can be customized by grade level, department, or other criteria.

**What if we need to cancel all trips?**
Administrators can cancel trips in bulk and automatically notify all affected parties.
