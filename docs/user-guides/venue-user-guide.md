# Venue User Guide

**TripSlip Venue App** - Complete guide for venue administrators

## Getting Started

### Account Setup

1. Visit `venue.tripslip.com`
2. Click "Sign Up" and create your account
3. Verify your email address
4. Complete your venue profile

### Venue Profile

Your venue profile includes:
- Venue name and description
- Address and contact information
- Operating hours
- Capacity and facilities
- Photos and media

## Creating Experiences

### Experience Setup

1. Navigate to "Experiences" in the sidebar
2. Click "Create New Experience"
3. Fill in the required information:
   - Experience name and description
   - Duration and capacity
   - Grade levels
   - Educational standards alignment
   - Pricing structure

### Pricing Configuration

- Set base price per student
- Add optional add-ons (lunch, materials, etc.)
- Configure group discounts
- Set minimum/maximum group sizes

### Availability Management

1. Go to "Calendar" section
2. Set available dates and time slots
3. Block out unavailable dates
4. Configure booking lead time

## Managing Bookings

### Viewing Bookings

- Dashboard shows all upcoming bookings
- Filter by date, status, or school
- View booking details and student count
- Download booking reports

### Booking Status

- **Pending**: Awaiting confirmation
- **Confirmed**: Booking confirmed
- **Completed**: Trip has occurred
- **Cancelled**: Booking cancelled

### Communication

- Send messages to teachers
- Receive booking notifications
- Update trip details
- Share important information

## Financial Management

### Stripe Connect Setup

1. Navigate to "Financial Settings"
2. Click "Connect Stripe Account"
3. Complete Stripe onboarding
4. Verify bank account information

### Payment Tracking

- View all payments received
- Track pending payments
- Download financial reports
- Export data for accounting

### Refund Processing

1. Go to booking details
2. Click "Issue Refund"
3. Select full or partial refund
4. Confirm refund amount
5. Refund processes automatically

## FAQ

**How do I get paid?**
Payments are automatically transferred to your bank account via Stripe Connect within 2-7 business days.

**Can I edit an experience after creating it?**
Yes, you can edit experiences at any time. Changes won't affect existing bookings.

**What if I need to cancel a booking?**
Contact the teacher as soon as possible. Refunds are processed according to your cancellation policy.

**How do I handle special requests?**
Communicate directly with teachers through the messaging system to accommodate special needs.

**Can I offer discounts?**
Yes, you can create promotional codes and group discounts in the pricing settings.
